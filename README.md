# 👻 Terminal Pac-Man Game in C++

A terminal-based Pac-Man inspired game developed in **C++** using Object-Oriented Programming principles. The game features player movement, enemy AI, collectible dots, score tracking, and collision detection within a text-based game environment.

---

## Features

- 🎮 Interactive terminal-based gameplay
- 👻 Ghost enemy with movement AI
- 🟡 Dot collection system
- 📈 Live score tracking
- 💀 Collision detection with ghosts
- 🧱 Maze with walls and obstacles
- ⌨️ Real-time keyboard controls
- 🧩 Modular Object-Oriented design

---

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Standard Template Library (STL)
- POSIX Terminal Libraries
  - `termios.h`
  - `unistd.h`

## Controls

| Key | Action |
|------|--------|
| W | Move Up |
| A | Move Left |
| S | Move Down |
| D | Move Right |
| Q | Quit Game |

---

## Gameplay

- Navigate Pac-Man through the maze.
- Collect dots to increase your score.
- Avoid the ghost while clearing the maze.
- The game ends if the ghost catches Pac-Man.
